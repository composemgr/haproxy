# Haproxy

A self-hosted haproxy application.

## Installation

### Option 1: Quick Install
```bash
curl -q -LSsf "https://raw.githubusercontent.com/composemgr/haproxy/main/docker-compose.yaml" | docker compose -f - up -d
```

### Option 2: Git Clone
```bash
git clone "https://github.com/composemgr/haproxy" ~/.local/srv/docker/haproxy
cd ~/.local/srv/docker/haproxy
docker compose up -d
```

### Option 3: Using composemgr
```bash
composemgr install haproxy
```
